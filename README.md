## 프로젝트 소개
 - 아이의 일상을 관리하고, 소중한 순간을 따뜻한 이야기로 기록하는 육아 지원 서비스입니다.
 - 가정통신문으로부터 준비물, 행사 등 잊기쉬운 일정을 OCR을 통해 추출하여 캘린더에 기록합니다.
 - LLM을 이용하여 아이의 하루를 요약하고, 키워드 바탕으로 맞춤형 동화를 생성합니다. 
 - 챗봇을 통해 아이와의 특별한 순간을 기록하고, 일기 작성을 도와줍니다.
